# Sign-Lingo: AI-Based Sign Language Tutor

Sign-Lingo is a comprehensive mobile educational platform designed to democratize American Sign Language (ASL) learning. Unlike passive video tutorials, Sign-Lingo utilizes real-time Computer Vision (MediaPipe) and Deep Learning (LSTM) to provide instant feedback on user gestures via a smartphone camera.

**Key Features:**
- **Real-Time AI Verification:** Detects 50+ dynamic ASL words and static alphabets using on-device processing.
- **Interactive Gamification:** Features daily streaks, XP rewards, and global leaderboards to boost retention.
- **Adaptive Learning:** A "Weakness Tracking Engine" identifies difficult signs and personalizes the curriculum.
- **Tech Stack:** React Native (Mobile), Python Flask (AI Backend), MongoDB (Database), and MediaPipe.

Sign-Lingo bridges the gap between expensive personal tutoring and self-study, making sign language accessible, interactive, and effective.
